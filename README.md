На интерфейсе расположены кнопка подключения к Arduino по UART, кнопка отключения и виджет QCustomPlot для отображения графика. После подключения по кнопке, от ардуино по последовательному порту каждую секунду приходит 2 байта с  рандомными числами от 0 до 255. Первый байт это координата по x, второй байт - по y.
 При нажатии на кнопку обновить график QCustomPlot отображает полученные точки.
